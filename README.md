Mini application with react.js. Codecademy clone.
=====================
Getting started
-----------------------------------
####Start JSON Server
```javascript
npm run server
```
json server is located on port 3001
####Start project
```javascript
npm start
```
####Admin panel
```html
/admin
```
The scope of the project
-----------------------------------
Frontend:
* react router

Admin panel:
* create new note
* edit note
* remove note
* sort
* filter
* search
* pagination
* form validation
